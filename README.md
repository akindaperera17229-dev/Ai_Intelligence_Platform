# Ai_Intelligence_Platform# AI Engineering Intelligence Platform

An AI-powered engineering workflow intelligence platform that analyzes development workflows, detects bottlenecks, and provides operational insights across engineering systems.

## Core Features

- Engineering workflow analysis
- Event-driven architecture
- AI-powered bottleneck detection
- GitHub activity intelligence
- Workflow risk analysis
- AI evaluation pipeline

## Tech Stack

### Backend
- Spring Boot
- PostgreSQL
- RabbitMQ

### AI Engine
- FastAPI
- OpenAI API

### Frontend
- React

## Architecture

Event-driven microservice architecture designed for engineering workflow intelligence and operational analysis.
